Magic Squares

A magic square is a square of n*n where n is the count of numbers in a row or column. All the numbers are distinct positive integers starting at 1. The numbers are organized such that each row, column, and the two main diagonals sum to the same number. The sum is called the magic constant. A square is said to have an order of n.

Magic Squares can be attempted at any order. However, it should be noted that the solution of a magic square at order 1 is trivial. There is a single solution of one number inside a single square. Also, at an order of 2 there are no solutions. With the numbers 1, 2, 3, and 4 we can see this in practice. There is no way for any number you pick to be added to two other numbers for the same result(1 + 2 = 3, 1 + 3 = 4, 1 + 4 = 5). Therefore it is unsolvable.

Solutions to a magic square are considered unique if they cannot be rotated or reflected to create another solution. This of course means that each solution has 8 semi-distinct squares. These 8 squares are considered equivalent and make up an Equivalence Class.

From this we can start to talk about distinct solutions. For a square of order 3 there is one distinct solution. For a square of order 4 there are 880 distinct solutions. For a square of order 5 there are over 200k solutions.
